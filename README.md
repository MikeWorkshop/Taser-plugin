Command: /taser get <taser | ricarica>

Cusom events: TaserEvent, RicaricaEvent

Use API: 
1- Add to pom.xml

	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>
  
2- Add the dependency

	<dependency>
	    <groupId>com.github.MikeWorkshop</groupId>
	    <artifactId>Taser-plugin</artifactId>
	    <version>Tag</version>
	</dependency>
